Wat je gaat doen

Web en Native

Web en native zijn de 2 belangrijkste vakken waarbij je leert over hoe je moet programmeren. Zoals je misschien al weet is web voor het maken van websites, en native voor het maken van windows apps.

Pratijk (Delta)

Praktijk is waar je projecten gaat maken. Van het oplossen van tickets tot het maken van een hele website! Hier leer je hoe je moet samenwerken, team verdelingen moet maken, een kanban bord gebruikt, enzv. Praktijk kan ook delta genoemt worden; daar zo meer over!

Overig

Natuurlijk hebben we hier ook gewoon de generic vakken zoals nederlands, engels, en rekenen. We krijgen ook een vak genaamd PRO. Daar programmeer je bijna niet, maar je leert wel allerlei manieren hoe software dev's projecten succesvol maken.

Het rooster voor de noobs

Ik bedoel 1ste jaars

Het rooster van het eerste jaar is best normaal. Het kan alleen veel verschillen. De ene dag moet je beginnen om 8:30, de ander om 11:00. Het hangt er helemaal vanaf in welke klas je zit.

Hiernaast zie je een rooster die echt HEEL gevarieerd is. Zoals ik al zei, soms kan je uitslapen, en soms moet je vroeg opstaan! De indeling is altijd anders. Het is gewoon rng bro.
